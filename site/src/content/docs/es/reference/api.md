---
title: "Referencia de la API"
description: "Todas las funciones y clases públicas de phonometry."
---

Toda la funcionalidad principal puede importarse directamente del paquete
`phonometry`.

| Nombre | Tipo | Descripción (entradas) | Uso (salidas) |
| :--- | :--- | :--- | :--- |
| `octavefilter` | `función` | **Análisis de alto nivel.**<br>• `x`: array de señal<br>• `fs`: frecuencia de muestreo [Hz]<br>• `fraction`: 1, 3, etc. (def.: 1)<br>• `order`: orden del filtro (def.: 6)<br>• `limits`: [f_min, f_max] (def.: [12, 20000])<br>• `filter_type`: 'butter', 'cheby1', 'cheby2', 'ellip', 'bessel' (def.: 'butter')<br>• `sigbands`: devolver señales por banda (def.: False)<br>• `detrend`: eliminar offset DC (def.: True)<br>• `calibration_factor`: multiplicador de sensibilidad (def.: 1.0)<br>• `dbfs`: salida en dBFS en lugar de dB SPL (def.: False)<br>• `mode`: 'rms' o 'peak' (def.: 'rms')<br>• `nominal`: etiquetas nominales IEC 61260-1 (def.: False)<br>• `show` / `plot_file`: gráfica de respuesta<br>• `ripple`: rizado de banda de paso [dB] (cheby1/ellip)<br>• `attenuation`: atenuación de banda atenuada [dB] (cheby2/ellip) | `spl, freq = octavefilter(x, fs, ...)`<br>• `spl`: niveles [dB]<br>• `freq`: frecuencias [Hz]<br><br>**Con `sigbands=True`:**<br>`spl, freq, xb = octavefilter(x, fs, sigbands=True)`<br>• `xb`: lista de señales filtradas (una por banda)<br><br>**Uso calibrado:**<br>`spl, f = octavefilter(x, fs, calibration_factor=0.05)` |
| `OctaveFilterBank` | `clase` | **Banco eficiente reutilizable.**<br>• `fs`: frecuencia de muestreo [Hz]<br>• `fraction`: 1, 3, etc.<br>• `order`: orden del filtro<br>• `limits`: [f_min, f_max] (def.: [12, 20000])<br>• `filter_type`: arquitectura<br>• `show` / `plot_file`: gráfica de respuesta<br>• `calibration_factor`: multiplicador de sensibilidad<br>• `dbfs`: usar dBFS (def.: False)<br>• `stateful`: conservar estado entre llamadas (def.: False)<br>• `steady_ic`: condiciones iniciales de régimen permanente (def.: False)<br>• `resample`: diezmado multitasa (def.: True)<br>• `ripple` / `attenuation`: parámetros de diseño [dB] | `bank = OctaveFilterBank(fs=48000, fraction=3, order=6, filter_type='butter')`<br>`spl, f = bank.filter(x, sigbands=False, mode='rms', detrend=True, zero_phase=False)`<br><br>• `bank.freq` / `bank.freq_d` / `bank.freq_u` / `bank.sos`: propiedades calculadas |
| `OctaveFilterBank.spectrogram` | `método` | **Niveles por banda vs tiempo.**<br>• `x`: array de señal (1D o 2D)<br>• `window_time`: longitud de ventana [s] (def.: 0.125)<br>• `overlap`: fracción en [0, 1) (def.: 0.5)<br>• `mode`: 'rms' o 'peak'<br>• `zero_phase`: ventanas sin retardo de grupo (def.: False) | `levels, freq, times = bank.spectrogram(x)`<br><br>• `levels`: (bandas, ventanas) o (canales, bandas, ventanas)<br>• `times`: centros de ventana [s] |
| `weighting_filter` | `función` | **Ponderación acústica.**<br>• `x`: array de señal<br>• `fs`: frecuencia de muestreo [Hz]<br>• `curve`: 'A', 'C', 'G' (infrasonido ISO 7196) o 'Z' (def.: 'A')<br>• `high_accuracy`: precisión clase 1 en HF por sobremuestreo interno (def.: True) | `y = weighting_filter(x, fs, curve='A')`<br><br>• `y`: señal ponderada |
| `WeightingFilter` | `clase` | **Filtro de ponderación reutilizable.**<br>• `fs`: frecuencia de muestreo [Hz]<br>• `curve`: 'A', 'C', 'G' o 'Z'<br>• `stateful`: procesado por bloques (def.: False)<br>• `steady_ic`: condiciones iniciales de régimen permanente (def.: False)<br>• `high_accuracy`: True por defecto salvo en stateful | `wf = WeightingFilter(fs, 'A')`<br>`y = wf.filter(x)` |
| `time_weighting` | `función` | **Captura de energía.**<br>• `x`: señal cruda (se eleva al cuadrado internamente; el tiempo es el último eje)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `mode`: 'fast', 'slow' o 'impulse'<br>• `initial_state`: None, 'zero', 'first', escalar o array (def.: None) | `env = time_weighting(x, fs, mode='fast')`<br><br>• `env`: envolvente de energía (valor cuadrático medio), misma forma que `x` |
| `TimeWeighting` | `clase` | **Ponderación temporal con estado.**<br>• `fs`: frecuencia de muestreo [Hz]<br>• `mode`: 'fast', 'slow', 'impulse' (def.: 'fast') | `tw = TimeWeighting(fs, mode='fast')`<br>`env = tw.process(block)` por bloque<br>`tw.reset()` para reiniciar |
| `leq` | `función` | **Nivel equivalente (Leq).**<br>• `x`: array de señal (1D o 2D)<br>• `calibration_factor`: multiplicador de sensibilidad (def.: 1.0)<br>• `dbfs`: salida en dBFS (def.: False) | `level = leq(x, calibration_factor=s)`<br><br>• `level`: escalar (1D) o array por canal (2D) |
| `laeq` | `función` | **Leq ponderado A (LAeq).**<br>• `x`: array de señal (1D o 2D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `calibration_factor` / `dbfs`: como `leq` | `level = laeq(x, fs, calibration_factor=s)`<br><br>• `level`: escalar (1D) o array por canal (2D) |
| `ln_levels` | `función` | **Niveles estadísticos (LN).**<br>• `x`: array de señal (1D o 2D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `n`: percentiles de excedencia (def.: (10, 50, 90))<br>• `mode`: 'fast', 'slow', 'impulse' (def.: 'fast')<br>• `weighting`: 'A', 'C', 'G', 'Z' o None (def.: None)<br>• `calibration_factor` / `dbfs`: como `leq` | `stats = ln_levels(x, fs, n=(10, 50, 90), weighting='A')`<br><br>• `stats`: dict `{10: L10, 50: L50, 90: L90}` |
| `lc_peak` | `función` | **Pico ponderado C (LCpeak).**<br>• `x`: array de señal (1D o 2D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `calibration_factor`: como `leq`<br>• `dbfs`: 0 dBFS = *pico* a fondo de escala (amplitud 1,0), a diferencia de la referencia RMS de `leq` | `peak = lc_peak(x, fs)`<br><br>• IEC 61672-1 §5.13; verificado contra la Tabla 5 |
| `sel` | `función` | **Nivel de exposición sonora (SEL/LAE).**<br>• `x`: señal del evento completo (1D o 2D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `weighting`: 'A', 'C', 'G', 'Z' o None (def.: None)<br>• `calibration_factor` / `dbfs`: como `leq` | `lae = sel(x, fs, weighting='A')`<br><br>• Nivel del evento normalizado a 1 s |
| `sound_exposure` | `función` | **Exposición sonora E (IEC 61252).**<br>• `x`: señal (1D o 2D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `duration_hours`: periodo de exposición representado (def.: duración de la grabación)<br>• `calibration_factor` | `E = sound_exposure(x, fs, duration_hours=8)`<br><br>• `E`: Pa²·h |
| `lex_8h` | `función` | **Exposición diaria LEX,8h / LEP,d.**<br>• Mismos parámetros que `sound_exposure` | `lex = lex_8h(x, fs, duration_hours=4)`<br><br>• Nivel normalizado a 8 h [dB] |
| `loudness_zwicker` | `función` | **Sonoridad de Zwicker (ISO 532-1:2017).**<br>• `x`: señal calibrada en Pa (1D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `field`: 'free' o 'diffuse' (def.: 'free')<br>• `stationary`: método estacionario del apartado 5 (def.: False)<br>• `calibration_factor`: unidades digitales a Pa (def.: 1.0) | `res = loudness_zwicker(x, fs)`<br><br>• `ZwickerLoudness`; los análisis variables en el tiempo añaden N5/N10 y la traza N(t) a 500 Hz |
| `loudness_zwicker_from_spectrum` | `función` | **Sonoridad estacionaria desde niveles de banda.**<br>• `levels`: 28 niveles de tercio de octava, 25 Hz–12,5 kHz [dB SPL]<br>• `field`: 'free' o 'diffuse' (def.: 'free') | `res = loudness_zwicker_from_spectrum(levels)`<br><br>• `res.loudness` [sonos], `res.loudness_level` [fonios] |
| `ZwickerLoudness` | `dataclass` | **Resultado de sonoridad.**<br>• `loudness`: N [sonos]<br>• `loudness_level`: LN [fonios]<br>• `specific`: N′(z), 240 bins de 0,1 Bark<br>• `n5` / `n10`: sonoridad percentil (solo variable en el tiempo)<br>• `time` / `loudness_vs_time`: traza a 500 Hz | `res.loudness, res.n5`<br><br>• Los campos temporales son `None` en resultados estacionarios |
| `sharpness_din` | `función` | **Sharpness en acum (DIN 45692).**<br>• `x`: señal (1D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `field`: 'free' o 'diffuse' (def.: 'free')<br>• `method`: 'din', 'aures' o 'bismarck' (def.: 'din')<br>• `calibration_factor`: unidades digitales a Pa (def.: 1.0) | `s = sharpness_din(x, fs)`<br><br>• `s`: sharpness [acum] |
| `sharpness_din_from_specific` | `función` | **Sharpness desde un patrón de sonoridad específica.**<br>• `specific`: N′(z), 240 valores a 0,1 Bark (p. ej. `ZwickerLoudness.specific`)<br>• `method`: 'din', 'aures' o 'bismarck' (def.: 'din') | `s = sharpness_din_from_specific(res.specific)`<br><br>• `s`: sharpness [acum] |
| `equal_loudness_contour` | `función` | **Isofónica ISO 226:2023.**<br>• `phon`: nivel de sonoridad, 20–90 fonos | `freqs, spl = equal_loudness_contour(40.0)`<br><br>• SPL en las 29 frecuencias preferentes |
| `loudness_level` | `función` | **Nivel de sonoridad de un tono puro (fonos).**<br>• `spl`: SPL del tono [dB]<br>• `frequency`: una de las 29 frecuencias de la Tabla 1 [Hz] | `phon = loudness_level(73.0, 63.0)` |
| `hearing_threshold` | `función` | **Umbral de audición (Tabla 1 de ISO 226).**<br>• (sin parámetros) | `freqs, tf = hearing_threshold()` |
| `tone_to_noise_ratio` | `función` | **Relación tono-ruido (ECMA-418-1 §11).**<br>• `x`: señal (1D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `tone_freq`: tono a evaluar [Hz] (def.: pico más alto)<br>• `resolution_hz`: resolución FFT (def.: 1.0) | `r = tone_to_noise_ratio(x, fs)`<br><br>• `ToneAssessment(frequency, ratio_db, criterion_db, prominent)` |
| `prominence_ratio` | `función` | **Relación de prominencia (ECMA-418-1 §12).**<br>• mismos parámetros que `tone_to_noise_ratio` | `r = prominence_ratio(x, fs, tone_freq=1000)`<br><br>• `ToneAssessment(...)` |
| `ToneAssessment` | `dataclass` | **Veredicto de prominencia tonal.**<br>• `frequency`: tono evaluado [Hz]<br>• `ratio_db`: TNR o PR [dB]<br>• `criterion_db`: límite de prominencia a esa frecuencia [dB]<br>• `prominent`: bool | `r = tone_to_noise_ratio(x, fs)`<br>`if r.prominent: ...` |
| `sti_from_impulse_response` | `función` | **STI completo, método indirecto (IEC 60268-16 Ed. 5).**<br>• `ir`: respuesta al impulso (1D)<br>• `fs`: frecuencia de muestreo [Hz] (>= 22500)<br>• `snr`: SNR [dB], escalar o 7 bandas (def.: None)<br>• `level`: 7 niveles de banda del habla [dB SPL]; activa enmascaramiento + umbral de recepción (def.: None)<br>• `ambient`: 7 niveles de banda de ruido [dB SPL]; requiere `level` | `res = sti_from_impulse_response(ir, fs, snr=25.0)`<br><br>• `STIResult` con `mtf` (7×14) |
| `stipa` | `función` | **Método directo STIPA (Anexo B).**<br>• `x`: señal STIPA grabada (1D), 15–25 s<br>• `fs`: frecuencia de muestreo [Hz] (>= 22500)<br>• `reference`: señal de la fuente medida en lugar del m = 0,55 nominal (def.: None)<br>• `level` / `ambient`: como `sti_from_impulse_response` | `res = stipa(recording, fs)`<br><br>• `STIResult` con `mtf` (7×2) |
| `stipa_signal` | `función` | **Generador de señal de prueba STIPA (A.4/A.6.1).**<br>• `fs`: frecuencia de muestreo [Hz] (>= 22500)<br>• `seconds`: duración [s] (def.: 18.0)<br>• `level_db`: nivel RMS [dB SPL] (def.: None → RMS 0.1)<br>• `seed`: semilla del ruido rosa (def.: None) | `sig = stipa_signal(48000, seconds=18.0)`<br><br>• Señal de prueba 1D, espectro masculino de la Ed. 5 |
| `STIResult` | `dataclass` | **Resultado del STI.**<br>• `sti`: 0 a 1<br>• `mti`: índices por banda (7,)<br>• `mtf`: valores m corregidos (7×14 o 7×2)<br>• `band_levels`: niveles usados o None<br>• `rating`: letra del Anexo F, 'A+'…'U' | `res = stipa(x, fs)`<br>`print(res.sti, res.rating)` |
| `sound_intensity` | `función` | **Intensidad sonora p-p (IEC 61043).**<br>• `p1`, `p2`: señales de los micrófonos [Pa], misma longitud<br>• `fs`: frecuencia de muestreo [Hz]<br>• `spacing`: separación entre micrófonos Δr [m]<br>• `rho`: densidad del aire (def.: 1.204)<br>• `c`: velocidad del sonido (def.: 343.0)<br>• `fraction`: None, 1 o 3 (def.: None)<br>• `limits`: [f_min, f_max] (def.: [12, 20000]) | `res = sound_intensity(p1, p2, fs, spacing=0.012, fraction=3)`<br><br>• `IntensityResult` |
| `IntensityResult` | `dataclass` | **Resultado de intensidad.**<br>• por banda (con `fraction`): `frequency`, `intensity` [W/m²], `intensity_level`, `pressure_level`, `pressure_intensity_index`, `direction` (±1), `bias_correction`<br>• banda ancha: equivalentes `total_*`<br>• `max_valid_frequency`: 0,1·c/Δr | `res.total_intensity_level`<br>`res.total_direction` |
| `field_indicators` | `función` | **Indicadores de campo del Anexo A de ISO 9614-1.**<br>• `pressure_levels`: Lpi por posición [dB]<br>• `normal_intensity`: Ini con signo por posición [W/m²] | `fi = field_indicators(lp, i_n)`<br><br>• `FieldIndicators(f2, f3, f4)` |
| `dynamic_capability_index` | `función` | **Capacidad dinámica Ld (ISO 9614-1 §3.12).**<br>• `pressure_residual_intensity_index`: δpI0 [dB]<br>• `bias_error_factor`: K [dB] (def.: 10.0) | `ld = dynamic_capability_index(18.0)`<br><br>• Adecuado cuando `Ld > F2` (criterio 1) |
| `CalibrationWarning` | `clase warning` | **Grabación de calibración poco fiable.**<br>La emite `calculate_sensitivity` — con `fs` presente y `validate=True` — cuando el tono es inestable (límite IEC 60942) o demasiado corto | `warnings.simplefilter("error", CalibrationWarning)` |
| `lden` | `función` | **Nivel día-tarde-noche (ISO 1996-1 §3.6.4).**<br>• `lday`/`levening`/`lnight`: LAeq por periodo [dB]<br>• `hours`: duración de periodos (def.: (12, 4, 8)) | `l = lden(63.2, 58.1, 51.4)` |
| `ldn` | `función` | **Nivel día-noche (ISO 1996-1 §3.6.5).**<br>• `lday`/`lnight`: LAeq por periodo [dB]<br>• `hours`: def. (15, 9) | `l = ldn(63.2, 51.4)` |
| `composite_rating_level` | `función` | **Evaluación compuesta de jornada (ISO 1996-1 §6.5).**<br>• `periods`: lista de (nivel_db, horas, ajuste_db) que suman 24 h | `r = composite_rating_level([(63, 12, 0), (58, 4, 5), (51, 8, 10)])` |
| `linkwitz_riley` | `función` | **Crossover de audio.**<br>• `x`: array de señal<br>• `fs`: frecuencia de muestreo [Hz]<br>• `freq`: frecuencia de cruce [Hz]<br>• `order`: cualquier número par (def.: 4) | `lo, hi = linkwitz_riley(x, fs, freq=1000, order=4)`<br><br>• `lo`: señal paso-bajo<br>• `hi`: señal paso-alto |
| `calculate_sensitivity` | `función`| **Calibración SPL.**<br>• `ref_signal`: señal de calibración<br>• `target_spl`: nivel del calibrador (def.: 94.0)<br>• `ref_pressure`: presión de referencia (def.: 20e-6) | `s = calculate_sensitivity(ref_signal, target_spl=94.0)`<br><br>• `s`: float (multiplicador a presión) |
| `verify_filter_class` | `función` | **Verificación de clase IEC 61260-1:2014.**<br>• `bank`: un `OctaveFilterBank`<br>• `num_points`: puntos de la malla de frecuencia por banda (def.: 32768) | `result = verify_filter_class(bank)`<br><br>• `result["overall_class"]`: 1, 2 o None<br>• `result["bands"]`: clase y márgenes por banda [dB] |
| `getansifrequencies` | `función` | **Generador de frecuencias ANSI.**<br>• `fraction`: 1, 3, etc. (obligatorio)<br>• `limits`: [f_min, f_max] (def.: [12, 20000]) | `f_cen, f_low, f_high, labels = getansifrequencies(fraction=3)`<br><br>• `f_cen` / `f_low` / `f_high`: centros y bordes [Hz]<br>• `labels`: etiquetas nominales IEC |
| `normalizedfreq` | `función` | **Frecuencias IEC estándar.**<br>• `fraction`: 1 o 3 | `freqs = normalizedfreq(fraction=3)`<br><br>• `freqs`: lista de frecuencias centrales estándar [Hz] |
| `sweep_signal` | `función` | **Excitación ESS (ISO 18233 Anexo B).**<br>• `fs`: frecuencia de muestreo [Hz]<br>• `f1`: frecuencia inicial [Hz]<br>• `f2`: frecuencia final [Hz] (≤ fs/2)<br>• `seconds`: duración [s]<br>• `amplitude`: pico (def.: 1.0)<br>• `fade`: fracción de medio Hann (def.: 0.01) | `s = sweep_signal(48000, 20, 20000, 3.0)`<br><br>• barrido sinusoidal exponencial 1D |
| `inverse_filter` | `función` | **Filtro inverso de Farina para un ESS.**<br>• Mismos parámetros que `sweep_signal` | `inv = inverse_filter(48000, 20, 20000, 3.0)`<br><br>• barrido invertido en el tiempo y compensado a +6 dB/oct |
| `impulse_response` | `función` | **Deconvolución del barrido (ISO 18233 B.5).**<br>• `recorded`: respuesta grabada (1D)<br>• `reference`: barrido emitido (1D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `method`: 'spectral' (def.) o 'farina'<br>• `f_range`: (f1, f2) para 'farina'<br>• `regularization`: término de Tikhonov (def.: 1e-6)<br>• `length`: muestras causales (def.: len(recorded))<br>• `return_full`: conservar la cola de distorsión (def.: False) | `ir = impulse_response(rec, sweep, fs)`<br><br>• respuesta al impulso 1D |
| `mls_signal` | `función` | **Secuencia de longitud máxima (ISO 18233 Anexo A).**<br>• `order`: longitud del registro N, 2-20 | `mls = mls_signal(16)`<br><br>• secuencia bipolar, longitud 2**N − 1 |
| `mls_impulse_response` | `función` | **Respuesta al impulso a partir de una MLS periódica.**<br>• `recorded`: respuesta que abarca periodos completos de la MLS (1D)<br>• `mls`: secuencia de excitación<br>• `length`: muestras de la respuesta al impulso (def.: 2**N − 1) | `ir = mls_impulse_response(rec, mls)`<br><br>• correlación cruzada circular / promediado |
| `decay_curve` | `función` | **Curva de decaimiento de Schroeder (ISO 3382-1 5.3.3).**<br>• `ir`: respuesta al impulso (1D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `band`: centro de banda [Hz] (def.: None → banda ancha)<br>• `fraction`: 1 o 3 (def.: 1) | `time, level = decay_curve(ir, fs)`<br><br>• `time` [s], `level` [dB] (0 dB en t=0) |
| `room_parameters` | `función` | **Acústica de salas (ISO 3382-1/2).**<br>• `ir`: respuesta al impulso (1D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `bands`: (f_min, f_max) o None (def.: (125, 4000))<br>• `fraction`: 1 o 3 (def.: 1) | `res = room_parameters(ir, fs)`<br><br>• `RoomAcousticsResult` por banda |
| `RoomAcousticsResult` | `dataclass` | **Parámetros de sala por banda.**<br>• `frequency`, `edt`, `t20`, `t30` [s]<br>• `c50`, `c80` [dB], `d50`, `ts` [s]<br>• `dynamic_range` [dB]<br>• `edt_valid`/`t20_valid`/`t30_valid`<br>• `curvature` [%] | `res.t30, res.c80, res.t30_valid`<br><br>• arrays con una entrada por banda |
| `open_plan_metrics` | `función` | **Métricas de oficinas diáfanas (ISO 3382-3).**<br>• `positions_m`: distancias a la fuente [m] (≥ 4)<br>• `spl_a_speech`: nivel de habla ponderado A por posición [dB]<br>• `sti_values`: STI por posición | `m = open_plan_metrics(r, lp, sti)`<br><br>• `OpenPlanResult` |
| `OpenPlanResult` | `dataclass` | **Números únicos de oficina diáfana.**<br>• `d2s`: tasa de decaimiento espacial [dB/duplicación]<br>• `lp_as_4m`: nivel de habla a 4 m [dB]<br>• `rd`: distancia de distracción [m]<br>• `rp`: distancia de privacidad [m] | `m.d2s, m.rd, m.rp`<br><br>• `nan` cuando no está definido |
| `airborne_insulation` | `función` | **Aislamiento a ruido aéreo en campo (ISO 16283-1).**<br>• `l1`/`l2`: niveles emisor/receptor [dB], 1D o (posiciones, bandas)<br>• `t2`: T de la sala receptora por banda [s]<br>• `area`: superficie del cerramiento S [m²] (para R')<br>• `volume`: V receptor [m³] (para R')<br>• `t0`: T0 de referencia [s] (def.: 0.5) | `ins = airborne_insulation(l1, l2, t2, area=10, volume=50)`<br><br>• `AirborneInsulationResult` |
| `weighted_rating` | `función` | **Índice ponderado + C/Ctr (ISO 717-1).**<br>• `values_by_band`: 16 tercios o 5 octavas [dB]<br>• `bands`: 'third-octave', 'octave' o None | `w = weighted_rating(R)`<br><br>• `WeightedRatingResult` (Rw, C, Ctr) |

## Notas

- `zero_phase=True` (en `OctaveFilterBank.filter` y `spectrogram`) filtra hacia
  delante y hacia atrás (`sosfiltfilt`): sin retardo de grupo, atenuación
  efectiva doble, solo análisis offline. Incompatible con `stateful=True`.
- `mode='peak'` incluye el transitorio de arranque del filtro; un tono que
  empieza de forma abrupta puede sobrepasar ~1 dB. Consulta
  [Calibración y dBFS](/phonometry/es/guides/calibration/).
- `octavefilter()` cachea internamente los diseños del banco (32 entradas), así
  que las llamadas repetidas con los mismos parámetros se saltan la fase de
  diseño. Para control explícito usa `OctaveFilterBank`.
