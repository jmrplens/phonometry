---
title: "Referencia de la API"
description: "Todas las funciones y clases públicas de phonometry."
---

Toda la funcionalidad principal puede importarse directamente del paquete
`phonometry`.

| Nombre | Tipo | Descripción (entradas) | Uso (salidas) |
| :--- | :--- | :--- | :--- |
| `octavefilter` | `función` | **Análisis de alto nivel.**<br>• `x`: array de señal<br>• `fs`: frecuencia de muestreo [Hz]<br>• `fraction`: 1, 3, etc. (def.: 1)<br>• `order`: orden del filtro (def.: 6)<br>• `limits`: [f_min, f_max] (def.: [12, 20000])<br>• `filter_type`: 'butter', 'cheby1', 'cheby2', 'ellip', 'bessel' (def.: 'butter')<br>• `sigbands`: devolver señales por banda (def.: False)<br>• `detrend`: eliminar offset DC (def.: True)<br>• `calibration_factor`: multiplicador de sensibilidad (def.: 1.0)<br>• `dbfs`: salida en dBFS en lugar de dB SPL (def.: False)<br>• `mode`: 'rms' o 'peak' (def.: 'rms')<br>• `nominal`: etiquetas nominales IEC 61260-1 (def.: False)<br>• `show` / `plot_file`: gráfica de respuesta<br>• `ripple`: rizado de banda de paso [dB] (cheby1/ellip)<br>• `attenuation`: atenuación de banda atenuada [dB] (def.: 72; cheby2/ellip — cheby2 requiere >= 70 dB para clase 1) | `spl, freq = octavefilter(x, fs, ...)`<br>• `spl`: niveles [dB]<br>• `freq`: frecuencias [Hz]<br><br>**Con `sigbands=True`:**<br>`spl, freq, xb = octavefilter(x, fs, sigbands=True)`<br>• `xb`: lista de señales filtradas (una por banda)<br><br>**Uso calibrado:**<br>`spl, f = octavefilter(x, fs, calibration_factor=0.05)` |
| `OctaveFilterBank` | `clase` | **Banco eficiente reutilizable.**<br>• `fs`: frecuencia de muestreo [Hz]<br>• `fraction`: 1, 3, etc.<br>• `order`: orden del filtro<br>• `limits`: [f_min, f_max] (def.: [12, 20000])<br>• `filter_type`: arquitectura<br>• `show` / `plot_file`: gráfica de respuesta<br>• `calibration_factor`: multiplicador de sensibilidad<br>• `dbfs`: usar dBFS (def.: False)<br>• `stateful`: conservar estado entre llamadas (def.: False)<br>• `steady_ic`: condiciones iniciales de régimen permanente (def.: False)<br>• `resample`: diezmado multitasa (def.: True)<br>• `ripple` / `attenuation`: parámetros de diseño [dB] (`attenuation` def.: 72; cheby2 requiere >= 70 dB para clase 1) | `bank = OctaveFilterBank(fs=48000, fraction=3, order=6, filter_type='butter')`<br>`spl, f = bank.filter(x, sigbands=False, mode='rms', detrend=True, zero_phase=False)`<br><br>• `bank.freq` / `bank.freq_d` / `bank.freq_u` / `bank.sos`: propiedades calculadas |
| `OctaveFilterBank.spectrogram` | `método` | **Niveles por banda vs tiempo.**<br>• `x`: array de señal (1D o 2D)<br>• `window_time`: longitud de ventana [s] (def.: 0.125)<br>• `overlap`: fracción en [0, 1) (def.: 0.5)<br>• `mode`: 'rms' o 'peak'<br>• `zero_phase`: ventanas sin retardo de grupo (def.: False) | `levels, freq, times = bank.spectrogram(x)`<br><br>• `levels`: (bandas, ventanas) o (canales, bandas, ventanas)<br>• `times`: centros de ventana [s] |
| `weighting_filter` | `función` | **Ponderación acústica.**<br>• `x`: array de señal<br>• `fs`: frecuencia de muestreo [Hz]<br>• `curve`: 'A', 'C', 'G' (infrasonido ISO 7196) o 'Z' (def.: 'A')<br>• `high_accuracy`: precisión clase 1 en HF por sobremuestreo interno (def.: True) | `y = weighting_filter(x, fs, curve='A')`<br><br>• `y`: señal ponderada |
| `WeightingFilter` | `clase` | **Filtro de ponderación reutilizable.**<br>• `fs`: frecuencia de muestreo [Hz]<br>• `curve`: 'A', 'C', 'G' o 'Z'<br>• `stateful`: procesado por bloques (def.: False)<br>• `steady_ic`: condiciones iniciales de régimen permanente (def.: False)<br>• `high_accuracy`: True por defecto salvo en stateful | `wf = WeightingFilter(fs, 'A')`<br>`y = wf.filter(x)` |
| `time_weighting` | `función` | **Captura de energía.**<br>• `x`: señal cruda (se eleva al cuadrado internamente; el tiempo es el último eje)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `mode`: 'fast', 'slow' o 'impulse'<br>• `initial_state`: None, 'zero', 'first', escalar o array (def.: None) | `env = time_weighting(x, fs, mode='fast')`<br><br>• `env`: envolvente de energía (valor cuadrático medio), misma forma que `x` |
| `TimeWeighting` | `clase` | **Ponderación temporal con estado.**<br>• `fs`: frecuencia de muestreo [Hz]<br>• `mode`: 'fast', 'slow', 'impulse' (def.: 'fast') | `tw = TimeWeighting(fs, mode='fast')`<br>`env = tw.process(block)` por bloque<br>`tw.reset()` para reiniciar |
| `leq` | `función` | **Nivel equivalente (Leq).**<br>• `x`: array de señal (1D o 2D)<br>• `calibration_factor`: multiplicador de sensibilidad (def.: 1.0)<br>• `dbfs`: salida en dBFS (def.: False) | `level = leq(x, calibration_factor=s)`<br><br>• `level`: escalar (1D) o array por canal (2D) |
| `laeq` | `función` | **Leq ponderado A (LAeq).**<br>• `x`: array de señal (1D o 2D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `calibration_factor` / `dbfs`: como `leq` | `level = laeq(x, fs, calibration_factor=s)`<br><br>• `level`: escalar (1D) o array por canal (2D) |
| `ln_levels` | `función` | **Niveles estadísticos (LN).**<br>• `x`: array de señal (1D o 2D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `n`: percentiles de excedencia (def.: (10, 50, 90))<br>• `mode`: 'fast', 'slow', 'impulse' (def.: 'fast')<br>• `weighting`: 'A', 'C', 'G', 'Z' o None (def.: None)<br>• `calibration_factor` / `dbfs`: como `leq` | `stats = ln_levels(x, fs, n=(10, 50, 90), weighting='A')`<br><br>• `stats`: dict `{10: L10, 50: L50, 90: L90}` |
| `lc_peak` | `función` | **Pico ponderado C (LCpeak).**<br>• `x`: array de señal (1D o 2D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `calibration_factor`: como `leq`<br>• `dbfs`: 0 dBFS = *pico* a fondo de escala (amplitud 1,0), a diferencia de la referencia RMS de `leq`<br>• `oversample`: recupera picos inter-muestra en HF (def.: 8; 1 = en rejilla, subestima hasta ~1,15 dB a 48 kHz) | `peak = lc_peak(x, fs)`<br><br>• IEC 61672-1 §5.13; verificado contra la Tabla 5 |
| `sel` | `función` | **Nivel de exposición sonora (SEL/LAE).**<br>• `x`: señal del evento completo (1D o 2D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `weighting`: 'A', 'C', 'G', 'Z' o None (def.: None)<br>• `calibration_factor` / `dbfs`: como `leq` | `lae = sel(x, fs, weighting='A')`<br><br>• Nivel del evento normalizado a 1 s |
| `sound_exposure` | `función` | **Exposición sonora E (IEC 61252).**<br>• `x`: señal (1D o 2D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `duration_hours`: periodo de exposición representado (def.: duración de la grabación)<br>• `calibration_factor` | `E = sound_exposure(x, fs, duration_hours=8)`<br><br>• `E`: Pa²·h |
| `lex_8h` | `función` | **Exposición diaria LEX,8h / LEP,d.**<br>• Mismos parámetros que `sound_exposure` | `lex = lex_8h(x, fs, duration_hours=4)`<br><br>• Nivel normalizado a 8 h [dB] |
| `loudness_zwicker` | `función` | **Sonoridad de Zwicker (ISO 532-1:2017).**<br>• `x`: señal calibrada en Pa (1D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `field`: 'free' o 'diffuse' (def.: 'free')<br>• `stationary`: método estacionario del apartado 5 (def.: False)<br>• `calibration_factor`: unidades digitales a Pa (def.: 1.0) | `res = loudness_zwicker(x, fs)`<br><br>• `ZwickerLoudness`; los análisis variables en el tiempo añaden N5/N10 y la traza N(t) a 500 Hz |
| `loudness_zwicker_from_spectrum` | `función` | **Sonoridad estacionaria desde niveles de banda.**<br>• `levels`: 28 niveles de tercio de octava, 25 Hz–12,5 kHz [dB SPL]<br>• `field`: 'free' o 'diffuse' (def.: 'free') | `res = loudness_zwicker_from_spectrum(levels)`<br><br>• `res.loudness` [sonos], `res.loudness_level` [fonios] |
| `ZwickerLoudness` | `dataclass` | **Resultado de sonoridad.**<br>• `loudness`: N [sonos]<br>• `loudness_level`: LN [fonios]<br>• `specific`: N′(z), 240 bins de 0,1 Bark<br>• `n5` / `n10`: sonoridad percentil (solo variable en el tiempo)<br>• `time` / `loudness_vs_time`: traza a 500 Hz | `res.loudness, res.n5`<br><br>• Los campos temporales son `None` en resultados estacionarios |
| `sharpness_din` | `función` | **Sharpness en acum (DIN 45692).**<br>• `x`: señal (1D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `field`: 'free' o 'diffuse' (def.: 'free')<br>• `method`: 'din', 'aures' o 'bismarck' (def.: 'din')<br>• `calibration_factor`: unidades digitales a Pa (def.: 1.0) | `s = sharpness_din(x, fs)`<br><br>• `s`: sharpness [acum] |
| `sharpness_din_from_specific` | `función` | **Sharpness desde un patrón de sonoridad específica.**<br>• `specific`: N′(z), 240 valores a 0,1 Bark (p. ej. `ZwickerLoudness.specific`)<br>• `method`: 'din', 'aures' o 'bismarck' (def.: 'din') | `s = sharpness_din_from_specific(res.specific)`<br><br>• `s`: sharpness [acum] |
| `loudness_moore_glasberg` | `función` | **Sonoridad de Moore-Glasberg (ISO 532-2:2017).**<br>• `x`: señal calibrada en Pa (1D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `field`: 'free', 'diffuse' o 'eardrum' (def.: 'free')<br>• `presentation`: 'binaural'/'diotic' o 'monaural' (def.: 'binaural') | `res = loudness_moore_glasberg(x, fs)`<br><br>• `MooreGlasbergLoudness` (patrón de excitación roex) |
| `loudness_moore_glasberg_from_spectrum` | `función` | **Sonoridad exacta desde componentes sinusoidales (ISO 532-2 §5.2/5.4).**<br>• `components`: secuencia de pares (frecuencia [Hz], nivel [dB SPL])<br>• `field`: 'free'/'diffuse'/'eardrum' (def.: 'free')<br>• `presentation`: (def.: 'binaural') | `res = loudness_moore_glasberg_from_spectrum([(1000.0, 40.0)])`<br><br>• 1 kHz/40 dB → 1.000 sono |
| `loudness_moore_glasberg_from_third_octave` | `función` | **Sonoridad desde 29 niveles de tercio de octava (ISO 532-2 §5.5).**<br>• `band_levels`: 29 niveles, 25 Hz–16 kHz [dB SPL]<br>• `field` / `presentation`: como arriba | `res = loudness_moore_glasberg_from_third_octave(levels)`<br><br>• `MooreGlasbergLoudness` |
| `MooreGlasbergLoudness` | `dataclass` | **Resultado de sonoridad de Moore-Glasberg.**<br>• `loudness`: N [sonos]<br>• `loudness_level`: LN [fonios]<br>• `specific`: N′(i), 372 bins de 0,1 Cam<br>• `erb_number` / `centre_frequencies`: ejes Cam / Hz<br>• `field` / `presentation`<br>• `.plot()`: sonoridad específica N′(i) sobre Cam | `res.loudness, res.loudness_level` |
| `loudness_moore_glasberg_time` | `función` | **Sonoridad variable en el tiempo (ISO 532-3:2023).**<br>• `signal`: Pa calibrada; 1D (diótica) o (n, 2) oídos<br>• `fs`: frecuencia de muestreo [Hz]<br>• `field`: 'free'/'diffuse'/'eardrum' (def.: 'free')<br>• `presentation`: (def.: 'binaural')<br>• `percentiles`: % de excedencia (def.: (1,5,10,50,90,95)) | `res = loudness_moore_glasberg_time(x, fs)`<br><br>• `MooreGlasbergTimeVaryingLoudness` |
| `MooreGlasbergTimeVaryingLoudness` | `dataclass` | **Resultado de sonoridad variable en el tiempo.**<br>• `time`: rejilla de 1 ms [s]<br>• `short_term_loudness` / `long_term_loudness`: S′(t)/S″(t) [sonos]<br>• `short_term_loudness_level` / `long_term_loudness_level` [fonios]<br>• `n_max` / `loudness_level_max`: sonoridad de largo plazo de pico<br>• `percentiles`: dict {porcentaje: sonos}<br>• `.plot()`: STL y LTL frente al tiempo | `res.n_max, res.percentiles[5.0]` |
| `loudness_ecma` | `función` | **Sonoridad del modelo de Sottek (ECMA-418-2:2025).**<br>• `signal_in`: señal calibrada en Pa (1D)<br>• `fs`: frecuencia de muestreo [Hz] (se remuestrea a 48 kHz)<br>• `field`: 'free' o 'diffuse' (def.: 'free') | `res = loudness_ecma(x, fs)`<br><br>• `EcmaLoudness`; 1 kHz/40 dB ≈ 1 sone_HMS |
| `EcmaLoudness` | `dataclass` | **Resultado de sonoridad de Sottek.**<br>• `loudness`: N [sone_HMS]<br>• `specific_loudness`: N′(z), 53 bandas Bark_HMS<br>• `bark` / `centre_frequencies`: ejes z / Hz<br>• `time` / `loudness_vs_time`: N(l) a 187,5 Hz<br>• `field`<br>• `.plot()`: N′(z) + N(l) | `res.loudness, res.specific_loudness` |
| `tonality_ecma` | `función` | **Tonalidad de Sottek (ECMA-418-2:2025).**<br>• `signal_in`: señal calibrada en Pa (1D)<br>• `fs`: frecuencia de muestreo [Hz] (se remuestrea a 48 kHz)<br>• `field`: 'free' o 'diffuse' (def.: 'free')<br>• `f_low` / `f_high`: banda de usuario opcional [Hz] (def.: None) | `res = tonality_ecma(x, fs)`<br><br>• `EcmaTonality`; 1 kHz/40 dB ≈ 1 tu_HMS |
| `EcmaTonality` | `dataclass` | **Resultado de tonalidad de Sottek.**<br>• `tonality`: T [tu_HMS]<br>• `specific_tonality`: T′(z), 53 bandas<br>• `tonal_frequencies`: f_ton,z por banda [Hz]<br>• `bark` / `centre_frequencies`<br>• `time` / `tonality_vs_time` / `tonal_frequency_vs_time`: T(l), f_ton(l)<br>• `field`<br>• `.plot()`: T′(z) + T(l) | `res.tonality, res.tonal_frequencies` |
| `roughness_ecma` | `función` | **Aspereza de Sottek (ECMA-418-2:2025), capacidad nueva.**<br>• `signal_in`: señal calibrada en Pa (1D)<br>• `fs`: frecuencia de muestreo [Hz] (se remuestrea a 48 kHz)<br>• `field`: 'free' o 'diffuse' (def.: 'free') | `res = roughness_ecma(x, fs)`<br><br>• `EcmaRoughness`; 1 kHz/70 Hz/100 %-AM/60 dB ≈ 1 asper |
| `EcmaRoughness` | `dataclass` | **Resultado de aspereza de Sottek.**<br>• `roughness`: R [asper] (percentil 90 de R(l50))<br>• `specific_roughness`: R′(z), 53 bandas<br>• `bark` / `centre_frequencies`<br>• `time` / `roughness_vs_time`: R(l50) a 50 Hz<br>• `specific_roughness_vs_time`: (n_times, 53)<br>• `field`<br>• `.plot()`: R(l50) + mapa de calor de aspereza específica | `res.roughness, res.specific_roughness` |
| `equal_loudness_contour` | `función` | **Isofónica ISO 226:2023.**<br>• `phon`: nivel de sonoridad, 20–90 fonos | `freqs, spl = equal_loudness_contour(40.0)`<br><br>• SPL en las 29 frecuencias preferentes |
| `loudness_level` | `función` | **Nivel de sonoridad de un tono puro (fonos).**<br>• `spl`: SPL del tono [dB]<br>• `frequency`: una de las 29 frecuencias de la Tabla 1 [Hz] | `phon = loudness_level(73.0, 63.0)` |
| `hearing_threshold` | `función` | **Umbral de audición (Tabla 1 de ISO 226).**<br>• (sin parámetros) | `freqs, tf = hearing_threshold()` |
| `tone_to_noise_ratio` | `función` | **Relación tono-ruido (ECMA-418-1 §11).**<br>• `x`: señal (1D)<br>• `fs`: frecuencia de muestreo [Hz]<br>• `tone_freq`: tono a evaluar [Hz] (def.: pico más alto)<br>• `resolution_hz`: resolución FFT (def.: 1.0) | `r = tone_to_noise_ratio(x, fs)`<br><br>• `ToneAssessment(frequency, ratio_db, criterion_db, prominent)` |
| `prominence_ratio` | `función` | **Relación de prominencia (ECMA-418-1 §12).**<br>• mismos parámetros que `tone_to_noise_ratio` | `r = prominence_ratio(x, fs, tone_freq=1000)`<br><br>• `ToneAssessment(...)` |
| `ToneAssessment` | `dataclass` | **Veredicto de prominencia tonal.**<br>• `frequency`: tono evaluado [Hz]<br>• `ratio_db`: TNR o PR [dB]<br>• `criterion_db`: límite de prominencia a esa frecuencia [dB]<br>• `prominent`: bool | `r = tone_to_noise_ratio(x, fs)`<br>`if r.prominent: ...` |
| `sti_from_impulse_response` | `función` | **STI completo, método indirecto (IEC 60268-16 Ed. 5).**<br>• `ir`: respuesta al impulso (1D)<br>• `fs`: frecuencia de muestreo [Hz] (>= 22500)<br>• `snr`: SNR [dB], escalar o 7 bandas (def.: None)<br>• `level`: 7 niveles de banda del habla [dB SPL]; activa enmascaramiento + umbral de recepción (def.: None)<br>• `ambient`: 7 niveles de banda de ruido [dB SPL]; requiere `level` | `res = sti_from_impulse_response(ir, fs, snr=25.0)`<br><br>• `STIResult` con `mtf` (7×14) |
| `stipa` | `función` | **Método directo STIPA (Anexo B).**<br>• `x`: señal STIPA grabada (1D), 15–25 s<br>• `fs`: frecuencia de muestreo [Hz] (>= 22500)<br>• `reference`: señal de la fuente medida en lugar del m = 0,55 nominal (def.: None)<br>• `level` / `ambient`: como `sti_from_impulse_response`<br>• avisa (`UserWarning`) si la grabación dura < 15 s (STI sesgado a la baja) | `res = stipa(recording, fs)`<br><br>• `STIResult` con `mtf` (7×2) |
| `stipa_signal` | `función` | **Generador de señal de prueba STIPA (A.4/A.6.1).**<br>• `fs`: frecuencia de muestreo [Hz] (>= 22500)<br>• `seconds`: duración [s] (def.: 18.0)<br>• `level_db`: nivel RMS [dB SPL] (def.: None → RMS 0.1)<br>• `seed`: semilla del ruido rosa (def.: None) | `sig = stipa_signal(48000, seconds=18.0)`<br><br>• Señal de prueba 1D, espectro masculino de la Ed. 5 |
| `STIResult` | `dataclass` | **Resultado del STI.**<br>• `sti`: 0 a 1<br>• `mti`: índices por banda (7,)<br>• `mtf`: valores m corregidos (7×14 o 7×2)<br>• `band_levels`: niveles usados o None<br>• `rating`: letra del Anexo F, 'A+'…'U' | `res = stipa(x, fs)`<br>`print(res.sti, res.rating)` |
| `sound_intensity` | `función` | **Intensidad sonora p-p (IEC 61043).**<br>• `p1`, `p2`: señales de los micrófonos [Pa], misma longitud<br>• `fs`: frecuencia de muestreo [Hz]<br>• `spacing`: separación entre micrófonos Δr [m]<br>• `rho`: densidad del aire (def.: 1.204)<br>• `c`: velocidad del sonido (def.: 343.0)<br>• `fraction`: None, 1 o 3 (def.: None)<br>• `limits`: [f_min, f_max] (def.: [12, 20000])<br>• `bias_correct`: corrige la subestimación por diferencias finitas de los totales de banda/banda ancha cerca de `max_valid_frequency` (def.: False) | `res = sound_intensity(p1, p2, fs, spacing=0.012, fraction=3)`<br><br>• `IntensityResult` |

| `field_indicators` | `función` | **Indicadores de campo del Anexo A de ISO 9614-1.**<br>• `pressure_levels`: Lpi por posición [dB]<br>• `normal_intensity`: Ini con signo por posición [W/m²] | `fi = field_indicators(lp, i_n)`<br><br>• `FieldIndicators(f2, f3, f4)` |

## Notas

- `zero_phase=True` (en `OctaveFilterBank.filter` y `spectrogram`) filtra hacia
  delante y hacia atrás (`sosfiltfilt`): sin retardo de grupo, atenuación
  efectiva doble, solo análisis offline. Incompatible con `stateful=True`. La
  banda de paso también se estrecha, reduciendo el nivel de banda ancha medido
  en ~0,2 a 0,3 dB por banda (un tono puro dentro de banda no se ve afectado);
  prefiere el filtrado hacia delante cuando el SPL absoluto de banda debe
  coincidir con el convenio de un solo paso, y reserva la fase cero para cuando
  importa la envolvente temporal (p. ej. decaimiento de reverberación).
- `mode='peak'` incluye el transitorio de arranque del filtro; un tono que
  empieza de forma abrupta puede sobrepasar ~1 dB. Consulta
  [Calibración y dBFS](/phonometry/es/guides/calibration/).
- `octavefilter()` cachea internamente los diseños del banco (32 entradas), así
  que las llamadas repetidas con los mismos parámetros se saltan la fase de
  diseño. Para control explícito usa `OctaveFilterBank`.
